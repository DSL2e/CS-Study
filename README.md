## 진행 방식 🌳(2주 한사이클)
* 2주마다 공부할 주제를 선정한다.
* CS 학습을 하고나서, MD형식으로 정리해서 스터디 전까지 PR을 올린다.
* 스터디 날에 자기가 공부한 CS 지식을 팀원들에게 발표하는 형식으로 진행한다.
* 발표 후, 1주동안 다른 팀원의 내용을 학습하면서 질문사항을 정리한다.
* 모든 주제의 학습이 끝나면, 다음 주에는 해당하는 주제로 CS 모의면접을 진행한다.
